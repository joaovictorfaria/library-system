# 📚 Library System

Sistema de gerenciamento de biblioteca desenvolvido com o objetivo de consolidar conhecimentos em desenvolvimento backend com C# e ASP.NET Core, arquitetura em camadas, SQL Server e Angular.

## Objetivos

Este projeto está sendo desenvolvido para simular um ambiente de desenvolvimento profissional, aplicando boas práticas de engenharia de software, documentação e versionamento.

Ao longo do desenvolvimento serão utilizados conceitos como:

- C#
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Angular
- Clean Architecture
- SOLID
- Design Patterns
- Dependency Injection
- JWT Authentication
- Git e GitHub
- Documentação Técnica
