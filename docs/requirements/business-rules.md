# Regras de Negócio

## Livros

- Um livro deve possuir título.
- Um livro deve possuir nome do autor.
- Um livro deve possuir ISBN.
- O ano de publicação deve ser válido.
- O ano de publicação não pode ser posterior ao ano atual.
- Um livro pode possuir vários exemplares.

## Exemplares

- Um exemplar só pode estar emprestado para um usuário por vez.

## Usuários

- Um usuário não pode exceder o limite de empréstimos ativos.

## Empréstimos

- Não é permitido realizar um empréstimo quando não houver exemplares disponíveis.
